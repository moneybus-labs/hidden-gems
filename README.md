## 🛠️ 승객 참여 가이드 (편집 방법)

이 페이지는 머니버스 승객이라면 누구나 개선 제안을 하거나 새로운 정보를 업데이트할 수 있습니다. 아래 순서에 따라 참여해 주세요.

### 1. 가장 빠른 참여 방법 (Fork 없이 바로 수정)

별도의 저장소 복사 없이 GitHub 웹에서 즉시 수정을 제안할 수 있습니다.

1. 수정하고 싶은 파일(예: `index.html`)을 클릭합니다.
2. 파일 우측 상단의 **연필 아이콘(Edit this file)**을 누릅니다.
3. 내용을 수정한 후(링크 추가, 오타 교정 등), 하단의 **Commit changes...** 버튼을 클릭합니다.
4. **"Create a new branch for this commit and start a pull request"** 옵션을 선택합니다. (자동 선택됨)
5. **Propose changes** 버튼을 누른 후, 다음 화면에서 **Create pull request**를 클릭하면 제안이 완료됩니다!

### 2. Fork를 이용한 참여 방법

1. 우측 상단의 **Fork** 버튼을 눌러 이 프로젝트를 본인의 저장소로 가져갑니다.
2. 본인의 저장소에서 코드를 수정하고 **Commit**합니다.
3. 원본 저장소(이곳)로 **Pull Request**를 보냅니다.
