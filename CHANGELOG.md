# 🚀 릴리즈 노트: '채상욱의 머니버스' 멀티링크

## [v1.0.4] - 2026-03-12

> **요약:** 우드워커 기여자 프로필 정보 업데이트 및 관심 ETF 목록 섹션 추가

### 🛠️ 수정 (Updates)
* **액티브 ETF 구성 변화 시각화 앱 링크 URL 변경:** 기존 Google Drive 링크를 단축 URL(`https://buly.kr/GvokGOa`)로 교체하였습니다.
* **우드워커 GitHub 프로필 이미지 연동:** 기여자 이름 옆에 GitHub(`joonhochoi`) 프로필 이미지가 표시되도록 개선되었습니다.

### ✨ 새로운 기능 (New Features)
* **관심 ETF 목록 섹션 추가:** 17계명과 링크 목록 사이에 관심 ETF를 카테고리별(미국 / 국내 / 고배당 / 중국)로 정리한 섹션이 추가되었습니다.
    * 각 ETF는 줄임말(닉네임) + 정식 상품명을 함께 표시하는 칩(chip) 형태로 구성되었습니다.
    * 브랜드(TIME / KoAct / KODEX / RISE / TIGER)별 색상 코딩이 적용되었습니다.
    * 편입 여부 미정인 ETF에는 '미정' 배지가 표시됩니다.

## [v1.0.3] - 2026-03-10

> **요약:** 신규 커뮤니티 링크 추가 및 링크 데이터 구조 리팩토링

### ✨ 새로운 기능 (New Features)
* **유틸리티 링크 추가:** 커뮤니티 기여(게임하는 판다 192331님)로 제작된 **'액티브 ETF 리포트'** 바로가기 링크가 추가되었습니다.

### 🛠️ 리팩토링 (Refactoring)
* **링크 데이터 분리:** 하드코딩되어 있던 링크 목록을 `data.js`의 `linksData` 배열로 이전하였습니다.
* **동적 렌더링:** `index.html`에 `renderLinks()` 함수를 추가하여, 링크 추가·수정 시 `data.js`만 편집하면 되도록 구조를 개선하였습니다.
